# AI Tic Tac Toe 🤖

An advanced Tic Tac Toe game built using HTML, CSS, and JavaScript featuring an AI opponent powered by Minimax Algorithm with Alpha-Beta Pruning and dynamic NxN board support.

---

## 🚀 Features

- 🧠 AI powered by Minimax Algorithm
- ⚡ Optimized using Alpha-Beta Pruning
- 🔄 Dynamic board size (3x3, 4x4)
- 🎮 Human vs AI gameplay
- 🎨 Modern UI with gradient design
- 🧩 Real-time decision making
- 📊 Mini decision visualization (debug mode)

---

## 📂 Project Structure
index.html
style.css
app.js


---

## 🧠 AI Logic

The AI uses:
- Minimax Algorithm
- Alpha-Beta Pruning for optimization
- Depth-based scoring system

This allows the AI to play optimally and never lose in standard conditions.

---

## 🎮 How to Play

1. Open `index.html`
2. Choose board size (3x3 or 4x4)
3. Play as **O**
4. AI plays as **X**
5. Try to beat the AI (good luck 😈)
